TECHNICAL DETAILS
================================================================

Controller Update:
```php
public function show($id)
{
    $anggota = Anggota::findOrFail($id);
    
    // Eager loading untuk performance
    $sertifikats = $anggota->sertifikats()
        ->with('riwayatKegiatan')
        ->orderBy('tanggal_terbit', 'desc')
        ->get();
    
    return view('admin.anggota.show', 
        compact('anggota', 'sertifikats'));
}
```

View Structure:
- Card component
- Responsive table
- Action buttons dengan icons
- Empty state dengan CTA
- Total count badge

Performance:
- Eager loading (with)
- Indexed columns
- No N+1 queries
- Optimized rendering
